Ably is a real-time service for syncing product data

1. Go to [Ably](https://www.ably.com/) and sign up for an account.
2. Create a new application if you don't already have one.
3. From the application, navigate to the `API Keys` tab.
4. Create a new key with the `Publish`, `Subscribe` and `Presence` permissions.
5. Set the following environment variables:

```bash
ABLY_API_KEY=<your-api-key>
```
